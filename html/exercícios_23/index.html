Agrupamento das tags com semântica

Tem dúvida sobre agrupamento das tags?

agrupamentos de tags é algo muito importante para a organização do nosso site, podemos agrupar as tags do jeito que quisermos isso é muito bom né, mais é ainda é muito melhor desenvolver os nossos sites com sentido. clique nos títulos!

header

cria áreas relativas a cabeçalho normalmente incluí títulos h1-h6, também pode conter menus de navegação

nav

define um conjunto de links de navegação.

.

main

O elemento define o conteúdo principal dentro do <body> em seu documento ou aplicação.
Nota: <main> não pode ser filho dos elementos <article>, <aside>, <footer>, <header>, ou <nav>. Autores não devem incluir mais de um elemento main no mesmo documento**.**

section

O elemento HTML <section> representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.

article

O Elemento HTML >article< representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuido de forma independente ou reutilizável, por exemplo, em sindicação.

aside

O elemento HTML <aside> representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo

footer

A tag footer diz respeito ao rodapé de nosso site, então quando quisermos criar um rodapé para nossa aplicação é importante usar a tag footer.
