símbolos

podemos inserir símbolos digitando &#codigo os códigos do HTML entites, pra ter os códigos basta pesquisar.
exemplo de símbolos;

© - copy right
€ - euro
® - registered trademark
¥ - yen
os simbolos também podem ser instalados usando &(nome do símbolo)
